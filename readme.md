# Prerequisites
  Java 8
  Maven 3.x
  
# Getting started
1. Launch the following command line on the root folder:
   *mvn clean install*
   
2. add the following dependency into the pom.xml file of the main application

```
    <dependency>
      <groupId>com.adneom.partition</groupId>
      <artifactId>partition</artifactId>
    </dependency>
```
